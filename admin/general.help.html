Networking

This sections allows you to configure how the server interacts with the network.

IPv6 enables the IPv6 support. If it is unselected the server will not use IPv6 even if the operating systems supports it.

Listen: IP Address to which the server will bind to wait for incoming requests.

Basic Behaviour

KeepAlive: enables the global keep alive support for the client requests. It increases the client performance by reusing the a connection for more than one request. It should most probably be turn on.

Server Permissions

If the server is launched with enough privileges, it is possible to change its effective user and group to prevent security issues. You can use either the user/group name or its numerical ID.

If a directory is specified as chroot, the server will jail itself inside it. It is important to know that, even if chroot is a security mechanism, it should not used as the only security measure.
